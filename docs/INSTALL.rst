wcc.audiofile Installation
--------------------------

To install wcc.audiofile using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.audiofile`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.audiofile

* Re-run buildout, e.g. with:

    $ ./bin/buildout
